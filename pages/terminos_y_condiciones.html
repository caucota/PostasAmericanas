El sitio no se responabiliza por los datos registrados ni el uso que cada integrante puede darles.

Cada usuario registrado acepta compartir de forma pública de forma gratuita tantos sus datos personales como las fotos o videos que se publiquen.

Cada usuario es responsable de su contraseña y sus datos registrados.

Cada usuario se compromete a hacer buen uso del sitio y de los datos, a mantener el respecto con todos y cada uno de los integrantes de este sitio.
